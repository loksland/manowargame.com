{% markdown %} #### Frequently Asked Questions Q. **I'm stuck on a level** {% endmarkdown %}
  • Balls move in unison, often forcing an undesired outcome. It may be useful to think about how to separate them.
  • Some levels require manipulating the balls in unconventional way. Keep experimenting and think outside the box.
  • If there is one difficult ball always left at the end it may help to devise the criteria for clearing this one and working back from there.
  • As a general rule of thumb the levels usually require logical solution rather than a skill challenge (This is not the case in the bonus stage!)
  • Red herrings are very rare. eg. If a cannon or platform exists it is usually part of the solution.
  • {% markdown %}If you're stuck tweet [@manowargame](https://twitter.com/manowargame) for a hint.{% endmarkdown %}
{% markdown %} Q. **Why is the last level locked?** Although you can attempt any level at any time, the final level only becomes playable when all other levels in the stage are completed.
